## 🏆 Highlight Projects

### 🏥 [Finger Rehabilitation](https://github.com/pecesool/Finger-rehabilitation) — Bachelor Thesis
**A cross-platform medical device system for stroke rehabilitation.**

- **Hardware**: Arduino Nano + 5× FSR pressure sensors + HC-05 Bluetooth — total cost under **$50**
- **Mobile App**: Flutter with Riverpod state management, real-time sensor visualization, SQLite persistence
- **AI Engine**: Automated report generation with linear regression, trend analysis, and personalized exercise recommendations — validated by Dr. Erzhan Rakhimovich at the Pavlodar Regional Cardiovascular Surgery Center
- **Clinical Study**: **30-day trial with 3 stroke patients** — all showed statistically significant improvement (p < 0.001). Patient with severe impairment improved **575%**
- **Thesis**: Sapienza University of Rome, supervised by Prof. Enrico Tronci

> *"The daily measurements provided evidence of improvement that contradicted the prognosis."* — Dr. Erzhan Rakhimovich
